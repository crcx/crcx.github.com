Software by Charles Childers

I mostly write code for my personal use, but a few of my programs have proven useful to others.

Lately I've been working exploring ways to make a simple email-only device aimed at non-techies into a significantly more useful platform. The results of this can be seen in Ent.

Ent has a non-public side as well, including an IRC-via-email client and additional information sources that are currently limited to things I personally find interesting. Maybe I'll make them public someday...
